# Miniproject — Diabetic Retinopathy Detection Platform

A full-stack medical diagnosis platform using Deep Learning (ResNet50) to detect Diabetic Retinopathy from retinal images.

## 🚀 Features
- **Auto-Training**: Automatically initiates model training upon dataset upload.
- **Supabase Integration**: Cloud-based PostgreSQL database for reliable storage.
- **Doctor Dashboard**: Upload images and view AI predictions with Grad-CAM visualization.
- **Organizer Dashboard**: Manage datasets, monitor training progress, and activate models.

## 🛠️ Tech Stack
- **Backend**: Flask (Python), SQLAlchemy, PyTorch.
- **Frontend**: React (Vite), CSS3.
- **Database**: Supabase (PostgreSQL).
